# 生產模擬與 STC 管制圖

使用 **SimPy** 進行離散事件模擬，比較兩種生產策略，並透過 **STC（Statistical Throughput Control，統計產出控制）** 管制圖進行視覺化分析。

---

## 📋 專案概述

本專案模擬生產環境，評估並比較兩種在固定時間內達成生產目標的策略：

- **加人力策略** — 在指定時間點增加額外工人以提升產能
- **加班策略** — 維持固定人數，允許超過正常工時繼續生產

透過 1,000 次 Monte Carlo 模擬進行統計分析，並以 STC 管制圖呈現結果。

---

## 🚀 功能特色

- ✅ 基於 SimPy 的離散事件模擬
- ✅ Monte Carlo 分析（達標率與產量完成率）
- ✅ 成本計算（含 1.5 倍加班費）
- ✅ 未在時限 `R` 內達標時自動外插估算完成時間
- ✅ STC 管制圖（5 條管制線，z = −1.675 至 1.675）
- ✅ 圖表標注關鍵模擬結果

---

## ⚙️ 參數設定

| 參數 | 數值 | 說明 |
|------|------|------|
| `R` | 9.8 小時 | 模擬總時長 |
| `quota` | 27,541 件 | 生產目標 |
| `prod_mean` | 235 件/小時 | 每人每小時平均產量 |
| `prod_variance` | 11 | 每人每小時產量變異數 |
| `wage` | $20.00/小時 | 每人時薪 |
| `initial_workers` | 11 人 | 初始工人數 |
| `T_add` | 2.4 小時 | 增援時間點 |
| `additional_workers` | 2 人 | 增加工人數 |
| `n_worker` | 11 人 | 加班策略固定工人數 |
| `mu` | 27,541 | STC 管制線 μ 參數 |
| `sigma` | 980 | STC 管制線 σ 參數 |

## 🔄 程式流程

```
設定參數
     │
     ▼
Monte Carlo 模擬（1,000 次）
     ├── 加人力策略 → 達標率 + 平均產量完成率
     └── 加班策略   → 達標率 + 平均產量完成率
     │
     ▼
單次模擬比較（固定隨機種子 seed=42，兩策略使用相同隨機序列）
     │
     ▼
成本計算（正常工資 + 1.5 倍加班費）
     │
     ▼
輸出報表 + 繪製 STC 管制圖（含結果標注）
```

---

## 💰 成本計算模型

**加人力策略**
- 初始工人：正常工時內按原薪計算，超過 `R` 小時以 1.5 倍計算
- 額外工人：從 `T_add` 開始計薪，超過 `R` 小時以 1.5 倍計算

**加班策略**
- 所有工人：正常工時內按原薪計算，超過 `R` 小時以 1.5 倍計算

---

## 📊 STC 管制圖說明

STC（Statistical Throughput Control，統計產出控制）管制圖繪製**實際累計產量與預定進度之差值**隨時間的變化，並疊加 z = −1.675、−0.674、0、0.674、1.675 五條管制線。

管制線公式：

```
x(t) = -((μ - Q) × (R - t) / R) - z × σ × √((R - t) / R)
```
